مثال‌های پردازش سیگنال - نسخه اصلاح شده

مثال مدولاسیون محصول (Product Modulation)

صورت سوال: فرض کنید \( x[n] = [1, 2, 3, 4, 5] \) و \( w[n] = [0.5, 1, 0.5, 0, 0] \) باشد. خروجی \( y[n] = w[n] \cdot x[n] \) را محاسبه کنید.

خروجی: \( y[n] = [0.5, 2, 1.5, 0, 0] \)

مثال جمع (Addition)

صورت سوال: فرض کنید \( x[n] = [1, 2, 3, 4, 5] \) و \( w[n] = [0.5, 1, 0.5, 0, 0] \) باشد. خروجی \( y[n] = w[n] + x[n] \) را محاسبه کنید.

خروجی: \( y[n] = [1.5, 3, 3.5, 4, 5] \)

مثال ضرب (Multiplication)

صورت سوال: فرض کنید \( x[n] = [1, 2, 3, 4, 5] \) و \( A = 2 \) باشد. خروجی \( y[n] = A \cdot x[n] \) را محاسبه کنید.

خروجی: \( y[n] = [2, 4, 6, 8, 10] \)

مثال معکوس زمانی (Time Reversal)

صورت سوال: فرض کنید \( x[n] = [0, 0, 1, 2, 3, 4, 5, 0, 0] \) برای \( n = -4 \) تا \( 4 \) باشد. خروجی \( y[n] = x[-n] \) را محاسبه کنید.

توضیح حل: برای معکوس زمانی، ایندکس‌ها نسبت به $ n = 0 $ قرینه می‌شوند. برای هر $ n $، مقدار $ x[-n] $ را پیدا کرده و در $ y[n] $ قرار می‌دهیم. مقادیر خارج از دامنه صفر فرض می‌شوند.

خروجی: $ y[n] = [0, 0, 5, 4, 3, 2, 1, 0, 0] $

مثال شاخه‌دهی (Branching)

صورت سوال: فرض کنید \( x[n] = [1, 2, 3, 4, 5] \) باشد. خروجی‌های شاخه‌ها را محاسبه کنید.

توضیح حل: در شاخه‌دهی، سیگنال بدون تغییر در چندین مسیر کپی می‌شود.

خروجی: هر دو خروجی = $ [1, 2, 3, 4, 5] $

مثال جابجایی زمانی (Time Shifting)

صورت سوال: فرض کنید \( x[n] = [0, 0, 1, 2, 3, 4, 5, 0, 0] \) برای \( n = -4 \) تا \( 4 \) باشد. $ y[n] = x[n - 2] $ و $ y[n] = x[n + 2] $ را محاسبه کنید.

توضیح حل: جابجایی زمانی شامل جابجا کردن ایندکس‌ها به چپ یا راست است. برای $ y[n] = x[n - 2] $، هر ایندکس 2 واحد به راست منتقل می‌شود. برای $ y[n] = x[n + 2] $، هر ایندکس 2 واحد به چپ منتقل می‌شود.

خروجی: Left Shift: $ [0, 0, 0, 0, 0, 1, 2, 3, 4] $, Right Shift: $ [1, 2, 3, 4, 5, 0, 0, 0, 0] $

مثال مقیاس‌بندی زمانی (Time Scaling)

صورت سوال: فرض کنید $ x[n] = [1, 2, 3, 4] $ برای $ n = 0 $ تا $ 3 $ باشد. $ y[n] = x[2n] $ و $ y[n] = x[n/2] $ را محاسبه کنید.

توضیح حل: در Downsampling ($ y[n] = x[2n] $)، هر دومین نمونه نگه داشته می‌شود. در Upsampling ($ y[n] = x[n/2] $)، بین نمونه‌ها صفر اضافه می‌شود.

خروجی: Downsampling ($ M = 2 $): $ [1, 3] $, Upsampling ($ N = 2 $): $ [1, 0, 2, 0, 3, 0, 4, 0] $

مثال عملیات آرایه‌ای (Array Operations)

صورت سوال: $ a[n] = \{1, 2, 4, -9\} $, $ b[n] = \{2, -1, 3, 3, 0\} $. $ c[n] = \{2 \cdot a[n] \cdot b[n]\} $, $ d[n] = \{a[n] + b[n]\} $, $ e[n] = 0.5 \cdot \{a[n]\} $ را محاسبه کنید.

خروجی: $ c[n] = \{4, -4, 24, -54, 0\} $, $ d[n] = \{3, 1, 7, -6, 0\} $, $ e[n] = \{0.5, 1, 2, -4.5\} $, Zero Padding: $ a[n] = \{1, 2, 4, -9, 0\} $, $ b[n] = \{2, -1, 3, 3, 0\} $

مثال جمع آرایه پرشده (Padded Array Addition)

صورت سوال: $ f[n] = \{-5, 2, -3\} $ برای $ 1 \leq n \leq 3 $, $ a[n] = \{1, 2, 4, -9\} $. $ g[n] = a[n] + f_{\text{pad}}[n] $ را محاسبه کنید.

خروجی: $ g[n] = \{1, -3, 6, -12\} $

مثال آرایه ترکیبی (Combined Array Example)

صورت سوال: $ a[n] = \{-3, 4, 2, -6, -9\} $ برای $-3 \leq n \leq 1$, $ b[n] = \{-3, -1, 0, 8, 7, -2\} $ برای $-2 \leq n \leq 3$, $ c[n] = \{1, 8, -3, 2, -6\} $ برای $3 \leq n \leq 7$. $ d[n] = a[-n + 3] $, $ e[n] = b[-n] $, $ f[n] = a[n] + b[-n + 2] + c[-n] $ را محاسبه کنید.

خروجی: $ d[n] = \{0, 0, -9, -6, 2, 4, -3\} $ برای $-3 \leq n \leq 3$, $ e[n] = \{-3, 0, 0, 8, -1, 0\} $ برای $-2 \leq n \leq 3$, $ f[n] = \{-4, 8, 8, 2, -10\} $ برای $-3 \leq n \leq 1$
